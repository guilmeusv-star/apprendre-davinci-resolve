# Apprendre DaVinci Resolve en 100 jours

Une application web progressive (PWA) pour documenter votre parcours d'apprentissage de DaVinci Resolve.

## Fonctionnalités

- 📱 Application web progressive (PWA)
- 🔄 Service Worker pour le mode hors ligne
- 🎨 Interface responsive

## Installation

1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
3. L'application s'installera comme PWA

## Technologies

- HTML5
- Service Workers
- Web App Manifest

Accédez [ici](https://guilmeusv-star.github.io/apprendre-davinci-resolve/) pour utiliser l'application.
